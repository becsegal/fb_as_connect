fb_as_connect by becarella


A Facebook authentication demo using the actionscript api.

Overview

If you're logged into facebook and have authorized the app, you should see your name and fb photo

If you're not logged into facebook or have not yet authorized the app, you should see a button to connect to facebook. Clicking the button opens a popup where you can login and/or authorize the app. The popup window will close when done and the swf will update automatically.

Also works as a Facebook app

More info @ becarella.com


Download


You can download the source in zip or tar format.

You can also clone the project with Git by running: 

$ git clone git://github.com/becarella/fb_as_connect
